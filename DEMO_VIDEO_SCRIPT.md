## Production Notes

### **Screen Recording Tips**

1. **Use High Resolution**: Record at 1920x1080 minimum
2. **Clean UI**: Use demo/staging environment with realistic but clean data
3. **Smooth Navigation**: Practice transitions before recording
4. **Highlight Cursor**: Use cursor highlighting tool for clarity
5. **Zoom on Key Actions**: Zoom in on important UI elements when needed

### **Voiceover Guidelines**

- **Pace**: Speak clearly, slightly slower than normal conversation
- **Tone**: Professional but warm, enthusiastic but not overly excited
- **Pauses**: Allow brief pauses after key demonstrations
- **Emphasis**: Emphasize key differentiators (AI learning, personalization, unified platform)

### **Visual Enhancements**

1. **Callouts**: Add text callouts for key features
2. **Arrows/Highlights**: Use subtle arrows to guide attention
3. **Transitions**: Use smooth fades between scenes
4. **Lower Thirds**: Add feature names as lower thirds when demonstrating

### **Music & Sound**

- **Background Music**: Subtle, professional, non-distracting
- **Volume**: Keep music at 20-30% of voiceover volume
- **Sound Effects**: Minimal, only for key actions (button clicks, notifications)
- **Fade Out**: Fade music during voiceover, bring up during transitions
